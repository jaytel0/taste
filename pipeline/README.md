# Pipeline

The reusable local pipeline is exposed through:

```bash
npm run taste
```

That command reads local reference images, calls `packages/ai`, and writes a
complete run folder under `.taste/runs/<run-id>/`.

This `pipeline/` directory contains notes and checked-in examples, not the
private images used to create them.

## Example

`pipeline/taste/04-skill/SKILL.md` is Jaytel's example taste skill generated by
this tool. It demonstrates the output format and level of specificity the
pipeline is intended to produce.
